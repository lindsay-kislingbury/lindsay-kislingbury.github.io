Time Tracker

Time tracker is a simple time-tracking tool for tracking and visualizing how you spend your time.

What I used:

node.js, express, mongoDB, mongoose, bcrypt, passport, jquery, bootstrap, dataTables,
javascript, html, css

Time Tracker Development

When I started this project, I intended to create a simple - not necessarily pretty - stopwatch that could create, update and delete data in a database.

I had very little web development experience, had never worked with node, express, or mongoDB.

As the project progressed, I eventually incorporated user sessions, data charts, tables, and a responsive interface.

Working on this project is the first time that I touched most of the technologies listed above. It was a learning curve, and I relied heavily on documentation and problem solving.

While I learned a lot about full stack web development - I learned even more about teaching myself how to use new systems and how to read documentation.

Read more about the process in my blog posts about the development!
