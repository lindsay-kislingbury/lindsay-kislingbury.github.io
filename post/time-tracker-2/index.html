Javascript Stopwatch

The stopwach is styled with boostrap and a little extra CSS for padding, sizing, and responsive breakpoints.

I originally ran into some problems with setInterval() . The timer was running too fast, and after doing some research, I found that relying on setInterval() for extact time down to the milliseconds wasn’t a good approach, I couldn’t completely rely on the browser.

I decided to write a function that checks for the current time every time I call setInterval(), if they are not equal, I subtract the difference.

Here’s a look at that function:

  function getShowTime(){
    updatedTime = new Date().getTime();
    if (savedTime){
        difference = (updatedTime - startTime) + savedTime;

Storing The Data

The model for a user includes an array of timestamp objects. The timestamp object includes a title, an array of tags, the date of creation, and the elapsed time. Currently, I am storing elapsed time with seperate hour, minute, second, and millisecond parts. I’m unsure how I will eventually want to use this data when I generate charts and analytics so this will likely change.

When the user clicks the save icon, an XMLHTTPRequest is created and posts the JSON data to a route which saves the data in the database.

time.js

function send(){
  var title = document.getElementById('title').value;
  var options = document.getElementById('tags').selectedOptions;
  var tags = Array.from(options).map(({value})=> value);
  var dateOptions = {
      month: "short",
      day: "2-digit",
      year: "numeric"
  }
  var time = {
      hours: hours,
      minutes: minutes,
      seconds: seconds,
      milliseconds: milliseconds
  }
  var date = new Date().toLocaleDateString('en-US', dateOptions);
  var xhr = new window.XMLHttpRequest();
  xhr.open('POST', '/time/stamp', true);
  xhr.setRequestHeader('content-type', 'application/json;charset=UTF-8');
  xhr.send(JSON.stringify({"title": title, "tags": tags, "date": date, "elapsedTime": time}));
}

TimeController.js

const stamp = async (req, res) => {
  console.log(req.body);
  const {title, tags, date, elapsedTime} = req.body;
  let timestamp = {
      title: title,
      tags: tags,
      date: date,
      elapsedTime: elapsedTime
  }
  let user = await User.findById(req.user.id);
  user.timestamps.push(timestamp);
  await user.save();
};

Responsive Design

While I didn’t want to spend very much energy on the design, I did want to create something that looks good and meets the basic requirements for a responsive and modern website.

I used a combination of Bootstrap 5.3, EJS, and CSS. I only used two @media rules: one for Desktop screens, and the other for mobile (including landscape).
